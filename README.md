# NeuroFlow - Personal Clarity Dashboard

A Next.js personal productivity application with mood tracking, habit management, decision journaling, and financial overview.

## Installation

```bash
# Install dependencies
npm install

# Install required packages
npm install three @react-three/fiber @react-three/drei gsap
npm install @paper-design/shaders-react
```

## Development

```bash
# Run development server on port 9002
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) in your browser.

## Features

- **Dashboard**: Cognitive map with focus score and 4 core areas
- **Mood Tracking**: Track and visualize emotional states
- **Habit Management**: Build and monitor daily habits
- **Decision Journal**: Document important decisions and insights
- **Financial Overview**: Track transactions and spending
- **Animated Backgrounds**: Three.js shader effects and grain gradients
- **Collapsible Sidebar**: Responsive navigation

## Tech Stack

- Next.js 15.5.9 (Turbopack)
- React 19.2.1
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Framer Motion
- Three.js with React Three Fiber
- @paper-design/shaders-react
